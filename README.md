# LeetCode 2 - Add Two Numbers (Adicionar Dois Números)

Este repositório contém a solução em **Java** para o problema clássico "Add Two Numbers" do LeetCode.

## 📝 Descrição do Problema

Você recebe duas listas encadeadas (Linked Lists) não vazias que representam dois números inteiros não negativos. Os dígitos são armazenados em **ordem reversa**, e cada um de seus nós contém um único dígito. O objetivo é somar os dois números e retornar a soma como uma lista encadeada.

Você pode assumir que os dois números não contêm nenhum zero à esquerda, exceto o próprio número 0.

### Exemplo
```text
Input:  l1 = [2 -> 4 -> 3], l2 = [5 -> 6 -> 4]
Output: [7 -> 0 -> 8]
Explicação: 342 + 465 = 807.
